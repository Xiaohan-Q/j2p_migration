## 🎥 录制技术方案

### 工具准备

1. **屏幕录制**:
   - OBS Studio (免费，专业)
   - Camtasia (付费，易用)
   - macOS: QuickTime Screen Recording

2. **终端美化**:
   - Windows Terminal / iTerm2
   - 主题：Dracula / Nord
   - 字体：Fira Code / Cascadia Code

3. **代码编辑器**:
   - VS Code
   - 主题：Dark+
   - 插件：代码高亮、minimap

4. **后期制作**:
   - DaVinci Resolve (免费)
   - Final Cut Pro / Premiere Pro

### 录制脚本

**预处理脚本** (放在 demo_video/ 目录下):

```bash
#!/bin/bash
# 准备演示环境

# 1. 清理历史输出
rm -rf emoji_migration/output/*
rm -rf emoji_migration/emoji_python

# 2. 重置终端
clear
echo "Java to Python Migration Tool - Demo"
echo "===================================="

# 3. 设置颜色和提示符
export PS1="\[\033[01;32m\]demo\[\033[00m\]:\[\033[01;34m\]\w\[\033[00m\]\$ "

# 4. 准备示例文件
cp examples/Calculator.java ./

# 完成
echo "✓ Demo 环境准备完成"
```
